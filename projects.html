My

Personal Website

Goal: I wanted to create a personal website where I could not only advertise myself and my achievements but also let other people keep track of my progress as an upcoming software engineer. Moreover, this was also a way to learn HTML, CSS, and basic JavaScript on my own and start learning web development.

Role/Responsibility: Since I was working alone, I had to think this out on my own and program it on my own.

Obstacles/Problems: I encountered a couple problems while building this website, namely having to learn three programming languages. My main obstacle with CSS was the box model and how to space all the elements out evenely... Another notable obstacle was the contact form. I had originally tried to learn PHP and include it as the backend to the contact form, however, I realized that I was hosting it with github, which only supported static websites. Therefore, I had to use formspree, which still works great. Lastly using javascript to have popup animations for the projects page was difficult. It took some time to understand and learn how to code the popups.
